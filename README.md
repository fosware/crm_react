# CRM - React
Customer Relationship Management (CRM)- React
#

## Build with
Node.js, Express, React, Vite, CSS, Tailwind CSS

## Topics
- React
- React Router Dom 6.4.3
- Consume Fake API with JSON Server

## Usage
```
git clone https://github.com/fosware/crm_react.git  

cd crm_react

npm install 

npm run dev
```
